# Daily Grind OS

A lightweight PWA to track your daily routine — works offline, installs on any phone.

## Features
- Build your own custom daily schedule
- Live clock with active block highlighting
- Tap to mark blocks done
- 7-day history with completion stats
- Streak counter
- Browser notifications when each block starts
- Fully offline (service worker cached)
- Installable on Android & iPhone

## How to use
1. Open the app and tap **Get Started**
2. Go to the **Edit** tab and add your time blocks
3. Set icon, label, time and category for each
4. Head to **Today** and start tracking

## Deploy your own
1. Fork this repo
2. Go to [netlify.com](https://netlify.com) → New site → Import from Git
3. Select this repo → Deploy
4. Open on your phone → Add to Home Screen
